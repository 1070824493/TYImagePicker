# TYImagePicker

一个精简的图片选择器的封装

## 版本及使用说明:

swift3.2 ->对应3.x版本

```
source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/1070824493/Specs.git'
pod 'TYImagePicker', '~> 3.x.x'
```

swift4 -> 对应4.x版本

```
source 'https://github.com/CocoaPods/Specs.git'
source 'https://github.com/1070824493/Specs.git'
pod 'TYImagePicker', '~> 4.x.x'
```
